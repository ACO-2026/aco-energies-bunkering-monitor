📡 Live AIS Integration Setup
📡
HOW IT WORKS: This dashboard calls your Netlify serverless function at /.netlify/functions/vessel-proxy. The function calls AISStream.io using your AISSTREAM_API_KEY environment variable stored securely on Netlify. Your key never appears in client-side code.
STEP 1 — Netlify Environment Variable
In Netlify → Project Configuration → Environment Variables → Add:
Key: AISSTREAM_API_KEY
Value: your_api_key

Then trigger a redeploy. Never put the key in HTML/JS.
STEP 2 — Dashboard Settings
Click ⚙ Settings in the header.
Confirm function URL: /.netlify/functions/vessel-proxy
Set refresh interval → Click Save & Test Connection.

If connection succeeds, status changes to LIVE.
⚙ Dashboard Settings — AIS Configuration
Configure your Netlify serverless proxy. Your live API key stays on Netlify server-side only.
